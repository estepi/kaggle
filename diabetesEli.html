Objetivo: Desarrollar un modelo de clasificacin para diabetes.

Datos: dataset descargado de la plataforma kaggle

Variables de la base:

  1. Number of times pregnant
  2. Plasma glucose concentration a 2 hours in an oral glucose tolerance test
  3. Diastolic blood pressure (mm Hg)
  4. Triceps skin fold thickness (mm)
  5. 2-Hour serum insulin (mu U/ml)
  6. Body mass index (weight in kg/(height in m)^2)
  7. Diabetes pedigree function
  8. Age (years)
  9. Class variable (0 or 1)
  • Class variable es la variable a predecir y toma los valores:
    • 0 si la persona no es diabtica
    • 1 si la persona es diabtica

Tipos de modelos para clasificar variables categricas:

  • Regresin logstica (simple o mltiple)
  • Decision tree -> library(rpart) para crear el rbol y library(rpart.plot) para visualizarlo
  • Random Forest -> library(randomForest)
  • Red neuronal -> library(neuralnet)

Pasos para el desarrollo del modelo:

  1. Cargar los datos
  2. Explorarlos (EDA: Exploraory Data Analysis)
  3. Separarlos en entrenamiento (TR) y testeo (TS)
  4. Crear el modelo con los datos de TR
  5. Calcular las predicciones para los 2 sets de datos (TR y TS)
  6. Validar

Modelo 1: Regresin Logstica

Mtodo de regresin que permite estimar la probabilidad de una variable cualitativa binaria en funcin de variables cuantitativas y/o categricas. Es decir, permite calcular la probabilidad de que la variable dependiente pertenezca a cada una de las dos categoras en funcin de los valores que adquieran las variables independientes.

  • Para hacerla se utiliza stat::glm() (generalized linear model).
  • La variable de salida debe ser factor.
  • Para ver como se relaciona la variable de salida con las variables de entrada se pueden visualizar los coeficientes de correlacin utilizando corrplot
# visualizacion de los coef de correlacion
